# La Hacienda Productos Gourmet

Sitio web para la presentación y venta de productos gourmet, diseñado con React y enfocado en escalabilidad, estética profesional y experiencia de usuario.

## 📖 Descripción del proyecto

El proyecto **La Hacienda Productos Gourmet** se ha desarrollado con un enfoque modular, aplicando principios inspirados en **SOLID** para garantizar mantenibilidad y escalabilidad.

### 🔹 Módulos en React (archivos `.jsx`)
Cada funcionalidad del sitio (productos, contacto, mapa, navegación) se implementa como un **componente independiente en React**.  
Este enfoque modular permite:
- Reutilización de componentes en diferentes secciones.
- Separación clara entre lógica y presentación.
- Escalabilidad al añadir nuevas pestañas o funcionalidades sin afectar el resto del sistema.

### 🔹 Módulos de estilos con CSS Modules
El proyecto utiliza **CSS Modules** para mantener estilos encapsulados y evitar conflictos entre clases.  
Ventajas:
- Consistencia visual en todas las secciones.
- Estilos aplicados únicamente al componente correspondiente.
- Facilidad para mantener y actualizar la apariencia del sitio.

## 🚀 Características principales
- **Diseño responsivo**: Adaptado a móviles, tablets y escritorio.
- **Secciones uniformes**: Productos, contacto, mapa y presentación con estilos consistentes.
- **Datos modulares en JSON**: Separación de contenido y presentación para fácil mantenimiento.
- **Integración de mapas**: Ubicación precisa y centrada de la tienda.
- **Preparado para escalabilidad**: Infraestructura pensada para e-commerce y logística futura.

## 🛠️ Tecnologías utilizadas
- **Frontend**: React, CSS Modules
- **Gestión de datos**: JSON estructurado
- **protipopado**: Figma
- **Herramientas de desarrollo**: React, Node.js
- **Lenguajes de progrmacion utilizados**: JavaScript
