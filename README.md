# productos_gourmet_la_hacienda

Sitio web para la presentación y venta de productos gourmet, diseñado con React y enfocado en escalabilidad, estética profesional y experiencia de usuario.

## 🚀 Características principales
- **Diseño responsivo**: Adaptado a móviles, tablets y escritorio.
- **Secciones uniformes**: Productos, contacto, mapa y presentación con estilos consistentes.
- **Datos modulares en JSON**: Separación de contenido y presentación para fácil mantenimiento.
- **Integración de mapas**: Ubicación precisa y centrada de la tienda.
- **Preparado para escalabilidad**: Infraestructura pensada para e-commerce y logística futura.

## 🛠️ Tecnologías utilizadas
- **Frontend**: React, CSS Modules
- **Gestión de datos**: JSON estructurado
- **protipopado**: Figma
- **Herramientas de desarrollo**: React, Node.js
- **Lenguajes de progrmacion utilizados**: JavaScript
